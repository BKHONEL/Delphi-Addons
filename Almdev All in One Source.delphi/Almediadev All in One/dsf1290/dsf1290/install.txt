For Delphi 5-XE3:

For C++Builder 5, 6:

 - open *.bpk file
 - install package

 - open *.cbproj file
 - install package

Note:

   if you use DynamicSkinForm VCL in Delphi 2007-XE3 or C++Builder 2007XE3
   you must open project sources in designing time and
   set Application.MainFormOnTaskBar to False!
